# Getting Started

Welcome to UAV Bible — the team's central knowledge base.

## Prerequisites

- Git installed
- Node.js >= 18
- Basic familiarity with Markdown

## Running Locally

```bash
# Clone the repo
git clone https://github.com/Aerotech-Innovations/uav-bible.git
cd uav-bible

# Install dependencies
npm install

# Start the dev server
npm run docs:dev
```

The site will be available at `http://localhost:5173/uav-bible/`.

## Building for Production

```bash
npm run docs:build
npm run docs:preview
```
